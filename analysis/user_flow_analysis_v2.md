# ユーザーフロー詳細分析（高解像度画像ベース）

## PoCユーザー（お客さん）フロー

### 前提条件
- **女の子とお店の連携**
- **SNSの連携**: LINE, X, Instagram, Googleアカウント, Appleアカウント
- **年齢制限**: 18歳以上（高校生不可）
- **個人情報取得許諾**

### メインフロー

#### 1. 初回登録フロー
1. **18歳以上（高校生不可）の確認**
   - 個人情報取得許諾
   - ファクター毎に属性登録を記載
   - 空欄検証窓口の記載

2. **LINEでログイン**
   - パスワードとログインが面倒と感じる

3. **診断機能・判断基準**
   - SNSアカウント登録
   - チャットボット・AIサイズ
   - 診断機能
   - 動画視聴・オファー受領
   - 診断結果表示とオファー受領

#### 2. 店舗選択フロー
1. **診断後の分岐**
   - 「ジャンルが好きですか？」
     - YES: 次のステップへ
     - NO: 人気店お勧め一覧へ

2. **店舗閲覧**
   - **あなたにあうお店！**: PoCマッチング（条件マッチング + 診断結果）
   - **料金の一覧**: 検索機能
   - **お店検索**: 条件検索
   - **マイページ**: 保存した店舗
   - **面接を希望する**: オファー一覧
   - **人気店お勧め一覧**

3. **店舗詳細確認**
   - 店舗情報の閲覧
   - 体験入店と面接を選択
   - 面接を希望する

#### 3. アピールポイント（ピンク付箋）
- **主婦でも上位層**
- **働きやすい環境、高収入**
- **フォロー一覧**
- **ニックネーム・連絡先**: 在籍期間とあったら行きたい店
- **仕事内容と店舗の雰囲気**

#### 4. 法的要件とゴール
- **「環境整備」という名目で候補店のリストを作成**
- **リストを手に入れることが目的**
- **JFAgamix**（システム名）
- **"条件選択" と" あなたの職場探しのリスト"**: 最新情報等提供窓口

### 登録フォーム項目
- **住居入力フォーム**
  - 氏名（入力項目）
  - 身分証・性別
  - プロフィール画像
  - 住所（都道府県）
  - 初勤務希望日
  - 要望・目標

## PoCユーザー（店長）フロー

### 前提条件
- **お店は情報の登録と閲覧が前提**

### メインフロー

#### 1. 店舗登録フロー
1. **チーム登録**
2. **女の子の一覧**: あなたのお店を選んでいる
3. **女の子を選択して面接希望する**

#### 2. 女の子検索・閲覧フロー
1. **女の子閲覧**
   - **MUSTとWANTで条件を表示**
   - 働のお店を希望している女の子のお店を見る

2. **条件検索とオファー受領**
   - 面接を希望する
   - オファーを送る

3. **店舗情報表示・管理画面**

#### 3. 店舗情報項目（求人）
以下の項目が含まれる（スプレッドシート形式で表示）：
- ケース1: 検索の女の子にオファーちゃん
- 判断基準の女の子たち「たとえばん」
- 判断基準（未定）: ガイドライン・業務内容・報酬額
- 店舗名、住所、電話番号
- 営業時間
- 給与
- MUST要件
- WANT要件
- アピールポイント
- 仕事内容
- 待遇
- 応募資格
- その他

**Google Sheetsリンク**: https://docs.google.com/spreadsheets/d/1YUEWePyFySJoRmyNECdLmMFEo5SSmXtSpCsiCL0E/edit?gid=0#gid=0

#### 4. 法的要件
- **店舗情報項目（求人）**
- **所属者件**
- **待遇条件**
- **年齢確認記録の保存**: 一緒に働きたい女の子を探す、店長のためにいるかどうかを判断する女の子のお店を見る

#### 5. 注意事項（ピンク付箋）
- **お店の情報を正す**: 募集しない場合は「イメクラ」を非表示で登録
- **URL等が正しい場合は追加記載しない**
- **相談窓口の設置**: 一緒に働きたくない場合は連絡ください
- **相談窓口の設置**: 一緒に働きたくない場合は連絡ください（重複）

## 主要な機能要素

### 1. AI診断機能
- AIチャットボットによる診断
- 診断結果に基づくランク判定
- 診断結果表示とオファー受領

### 2. マッチング機能
- **PoCマッチング**: 条件マッチング + 診断結果
- **MUST要件とWANT要件**の判定
- 店舗とキャストの相互マッチング

### 3. コミュニケーション機能
- オファー送信・受領
- 面接希望の送信
- チャット機能（推測）

### 4. 法的コンプライアンス
- 18歳以上（高校生不可）の確認
- 個人情報取得許諾
- 苦情処理窓口の記載
- 年齢確認記録の保存
- 相談窓口の設置

### 5. システム名
- **JFAgamix**
