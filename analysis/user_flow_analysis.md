# ユーザーフロー分析（画像ベース）

## PoCユーザー（お客さん）フロー

### 1. 登録・認証
- **女の子とお店の連携**
  - SNSの連携: LINE, X, Instagram, Googleアカウント, Appleアカウント
  - 18歳以上（高校生不可）
  - 個人情報取得許諾

### 2. 登録フロー
1. **LINEでログイン**
2. **初回登録（お客さま不可）**
   - ファクター選択/情報登録
   - ファクター毎に属性登録
   - 空欄検証窓口の記載
3. **診断機能**
   - チャットボット・AIサイズ
   - 診断機能
   - 動画視聴・オファー受領
   - 診断結果表示とオファー受領

### 3. 診断後のフロー
- **ジャンルが好きですか？**
  - YES: 次のステップへ
  - NO: 人気店お勧め一覧

### 4. マッチングフロー
- **店舗の一覧**
  - あなたにあうお店！
  - PoCマッチング（条件マッチングと診断結果の組み合わせ）
  - 検索機能
- **条件検索**
  - 料金の一覧
  - 体験入店と面接を選択
  - 面接を希望する
  - オファー一覧

### 5. アピールポイント
- 主婦でも上位層
- 働きやすい環境、高収入
- フォロー一覧

### 6. ニックネーム・連絡先
- 在籍期間とあったら行きたい店
- 仕事内容と店舗の雰囲気

### 7. 最終確認
- 「環境整備」という名目で候補店のリスト
- 最新情報等提供窓口

### 8. 法的要件
- リストを手に入れることが目的
- JFAgama
