# 主要画面ワイヤーフレーム

## 1. LP / 店舗一覧ページ（ゲスト・会員共通）

### 目的
- **ゲスト**: 豊富な店舗情報を提示し、サービスの魅力を伝える。
- **会員**: パーソナライズされた情報を提供し、エンゲージメントを高める。

### 構成要素

| 要素 | 説明 |
| :--- | :--- |
| **ヘッダー** | サービスのロゴ。会員の場合は「通知」「マイページ」へのリンクを追加。 |
| **パーソナライズエリア（会員のみ）** | 「あなたへのおすすめ」「新着オファー」などを表示するセクション。 |
| **店舗カード** | 店舗の写真、店名、エリア、時給、アピールポイント（タグ形式）を表示。 |
| **無限スクロール** | ユーザーがスクロールすると、次々と店舗カードが読み込まれる。 |

### レイアウト（テキスト表現）

```
+----------------------------------------------------+
| [ロゴ]                              [通知][マイページ] (←会員のみ) |
+----------------------------------------------------+
|                                                    |
|    +------------------------------------------+    |
|    |         パーソナライズエリア             |    | (←会員のみ)
|    +------------------------------------------+    |
|                                                    |
|    +------------------+ +------------------+    |
|    |      店舗カード1     | |      店舗カード2     |    |
|    +------------------+ +------------------+    |
|                                                    |
|    +------------------+ +------------------+    |
|    |      店舗カード3     | |      店舗カード4     |    |
|    +------------------+ +------------------+    |
|                                                    |
|                  (無限スクロール)                  |
|                        ・                          |
|                        ・                          |
|                        ・                          |
+----------------------------------------------------+
```

## 2. 店舗詳細ページ

### 目的
- 店舗の魅力を詳細に伝え、ユーザーの興味を具体的なアクションに繋げる。

### 構成要素

| 要素 | 説明 |
| :--- | :--- |
| **メイン画像** | 店舗の雰囲気が伝わる大きな写真やスライドショー。 |
| **基本情報** | 店名、エリア、時給、待遇、募集要項など。 |
| **アピールポイント** | タグや文章で、お店の強みや特徴を訴求。 |
| **CTAボタン** | 「仮面接を受けてみる」「オファーを送る」など、次のアクションを促すボタン。画面下部に固定表示するのも有効。 |

### レイアウト（テキスト表現）

```
+----------------------------------------------------+
| ←戻る                                              |
+----------------------------------------------------+
|                                                    |
|    +------------------------------------------+    |
|    |                メイン画像                  |    |
|    +------------------------------------------+    |
|                                                    |
|    <h2>店舗名</h2>                                  |
|    <p>エリア / 時給: XXXX円</p>                      |
|                                                    |
|    <h3>アピールポイント</h3>                          |
|    <p>[タグ1] [タグ2] [タグ3]</p>                     |
|                                                    |
|    <h3>募集要項</h3>                                  |
|    <p>...................................</p>     |
|                                                    |
+----------------------------------------------------+
| [仮面接を受けてみる] [オファーを送る] (←画面下部固定) |
+----------------------------------------------------+
```

## 3. LINEログインモーダル

### 目的
- ユーザーにストレスなく、LINEログインを完了してもらう。

### 構成要素

| 要素 | 説明 |
| :--- | :--- |
| **見出し** | 「LINEでログインして、さらに便利な機能を使おう！」など、メリットを提示。 |
| **説明文** | ログインすることで何ができるようになるかを簡潔に説明。 |
| **LINEログインボタン** | LINEの公式ログインボタン。 |
| **閉じるボタン** | モーダルを閉じ、元の画面に戻るためのボタン。 |

### レイアウト（テキスト表現）

```
          +----------------------------------+
          |         LINEではじめる           |
          |                                  |
          |   ログインすると、お店からの     |
          |   特別なオファーが届きます。     |
          |                                  |
          |   +--------------------------+   |
          |   |    LINEでログイン/登録     |   |
          |   +--------------------------+   |
          |                                  |
          |            閉じる              |
          +----------------------------------+
```
