# SNSログイン機能 実装可能性調査レポート

## 1. 調査目的

本レポートは、キャバクラ面接アプリにおいて、ユーザーの利便性向上を目的としたLINEおよびInstagramによるログイン機能の実装可能性を評価し、その結果を報告するものです。

## 2. 調査サマリー

| SNS | 実装可能性 | 結論・推奨事項 |
| :--- | :--- | :--- |
| **LINE Login** | ✅ **実装可能** | ターゲットユーザーとの親和性が高く、実装も容易なため、**強く推奨**します。 |
| **Instagram Login** | ❌ **実装不可** | 一般ユーザー向けのAPIが2024年12月に廃止されたため、**実装できません**。 |
| **Facebook Login** | ✅ **実装可能** | Instagramの代替として有効であり、実装も容易なため、**推奨**します。 |

## 3. 各SNSログイン機能の詳細評価

### 3.1. LINE Login

- **実装可能性**: **可能**です。OAuth 2.0およびOpenID Connectに準拠した標準的なAPIが提供されており、公式ドキュメントやSDKも充実しています。
- **ユーザー識別**: 認証時に取得できる`userId`は、チャネルごとに発行される一意かつ不変のIDであり、ユーザーの識別に利用できます。
- **取得可能情報**: `profile`スコープでユーザーID、表示名、プロフィール画像URLが取得できます。別途申請すれば`email`スコープでメールアドレスも取得可能です。
- **メリット**:
  - 日本国内、特に若年層の利用率が非常に高い。
  - ユーザーは使い慣れたLINEアカウントで簡単に登録・ログインできるため、コンバージョン率の向上が期待できる。
- **デメリット**: 特にありません。

### 3.2. Instagram Login

- **実装可能性**: **不可能**です。一般ユーザーが自身のInstagramアカウントでサードパーティアプリにログインするための「Instagram Basic Display API」は、**2024年12月4日をもって廃止**されました。
- **代替API**: 現在提供されているInstagram関連のAPI（`Instagram API with Instagram Login`など）は、ビジネスアカウントまたはクリエイターアカウントのみを対象としており、本アプリのターゲットユーザー（一般の女子大生など）には適していません。
- **結論**: Instagramアカウントを利用した直接のログイン機能は実装できません。

### 3.3. Facebook Login

- **実装可能性**: **可能**です。LINE同様、OAuth 2.0ベースのAPIとJavaScript SDKが提供されており、実装は比較的容易です。
- **ユーザー識別**: 認証時に取得できる`userID`は、アプリごとに発行される一意のIDであり、ユーザー識別に使用できます。
- **取得可能情報**: `public_profile`スコアでユーザーID、氏名、プロフィール画像が、`email`スコープでメールアドレスが取得できます。
- **メリット**:
  - Instagramの代替手段として有効です。多くのユーザーがFacebookとInstagramのアカウントを連携させているため、実質的にInstagramユーザーを取り込むことが可能です。
  - グローバルでの利用者数が多く、将来的な海外展開の際にも活用できます。
- **デメリット**: LINEと比較すると、日本の若年層におけるアクティブ率はやや低い可能性があります。

## 4. 結論と推奨事項

以上の調査結果から、キャバクラ面接アプリのSNSログイン機能として、以下の実装を推奨します。

**推奨する実装**

1.  **LINE Login**: 必須機能として実装することを強く推奨します。
2.  **Facebook Login**: Instagramログインの代替として、またより多くのユーザーをカバーするために実装を推奨します。

この2つのSNSログインを実装することで、ターゲットユーザーの大部分をカバーし、登録時のハードルを大幅に下げることが期待できます。
