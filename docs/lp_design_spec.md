# キャバクラ面接アプリ LPデザイン仕様書

## 1. 概要

本ドキュメントは、キャバクラで働きたいと考える若い女性（特に大学生や未経験者）をターゲットとしたランディングページ（LP）のデザイン仕様を定義するものです。

## 2. デザインコンセプト

**「新しいワタシ、見つかるかも。」**

夜職への不安を払拭し、新しいキャリアの可能性や、自分らしく輝ける場所を見つけられる期待感を醸成する。プロフェッショナルでありながらも、親しみやすく、安心感のあるデザインを目指す。

## 3. ターゲットユーザー

- 18歳〜25歳の女性
- 大学生、フリーター
- 夜職未経験者または経験の浅い層
- 副業や短期での高収入に関心がある

## 4. デザインの方向性

- **トーン＆マナー**: 明るく、清潔感があり、少し大人っぽい。女性ファッション誌のような洗練された雰囲気を意識する。
- **カラー**: 
  - **メインカラー**: #FFFFFF (白) を基調とし、クリーンな印象を与える。
  - **アクセントカラー**: #FF69B4 (ホットピンク) や #FFC0CB (ピンク) などの女性らしい色を控えめに使用し、華やかさをプラス。信頼感を与えるために、#4A90E2 (落ち着いた青) も部分的に使用する。
- **タイポグラフィ**: 
  - **フォント**: 可読性が高く、柔らかい印象のゴシック体（例: Noto Sans JP）を基本とする。
  - **サイズ**: メリハリをつけ、重要なメッセージが目に留まるように調整する。
- **写真・イラスト**: 
  - **写真**: 実際に働く女性の写真は使用せず、ライフスタイルやファッションを想起させる、顔がはっきり映らないイメージカットを使用する。過度な露出や派手なメイクは避ける。
  - **イラスト**: シンプルでモダンな線画イラストを使用し、サービスの仕組みやメリットを分かりやすく伝える。

## 5. ページ構成案

| セクション | 内容 | デザイン・UI/UXのポイント |
| :--- | :--- | :--- |
| **ファーストビュー** | - **キャッチコピー**: 「新しいワタシ、見つかるかも。」<br>- **サブコピー**: 「カンタンな質問に答えるだけで、あなたにピッタリのお店が見つかるかも。まずは無料診断から。」<br>- **CTAボタン**: 「無料で適性診断をはじめる」 | - 背景には、夜景やおしゃれな街並みのイメージ画像を薄く配置。<br>- キャッチコピーは大きく、中央に配置して視線を集める。<br>- CTAボタンは最も目立つ色（アクセントカラー）で、クリックしやすいサイズにする。 |
| **共感・課題提起** | - 「こんな悩み、ありませんか？」<br>  - 「自分に合うお店がわからない」<br>  - 「ノルマや人間関係が不安」<br>  - 「学業やプライベートと両立したい」 | - 悩みを箇条書きにし、共感を誘う。<br>- アイコンやイラストを使い、視覚的に分かりやすくする。 |
| **ソリューション** | - **見出し**: 「Nightwork AIが、あなたの不安を解消します。」<br>- **3つのポイント**: <br>  1. **AIによる適性診断**: あなたの性格や希望から、最適な働き方を提案。<br>  2. **安心の店舗のみ掲載**: 厳しい審査をクリアした優良店だけ。<br>  3. **匿名での相談OK**: 個人情報を登録する前に、気になるお店と匿名でやりとりできる。 | - 各ポイントを大きなカード形式でデザインし、それぞれにアイコンと簡潔な説明文を添える。<br>- 「AI」や「匿名」といったキーワードを強調し、安心感と先進性をアピールする。 |
| **利用者の声** | - 3名程度の架空の利用者の声を紹介。<br>  - 例：「大学と両立しながら、無理なく働けています！」（20歳・大学生）<br>  - 例：「未経験だったけど、サポートの厚いお店に出会えました。」（22歳・フリーター） | - 顔写真は使わず、年代と属性がわかるアイコンと吹き出しで表現する。<br>- ポジティブで具体的な体験談を掲載し、信頼性を高める。 |
| **ご利用の流れ** | 1. **無料診断**: LINEでカンタンな質問に答える。<br>2. **結果確認**: あなたに合うお店のタイプがわかる。<br>3. **お店と話す**: 気になるお店と匿名でチャット。<br>4. **面接へ**: 条件が合えば面接へ。 | - ステップを番号付きのリストとイラストでシンプルに図解する。<br>- 各ステップが簡単であることを強調する。 |
| **よくある質問** | - Q. 本当に無料ですか？<br>- Q. 個人情報は安全ですか？<br>- Q. 無理な勧誘はありませんか？ | - アコーディオン形式で実装し、クリックすると回答が表示されるようにする。<br>- ユーザーの不安を先回りして解消する。 |
| **クロージング** | - **見出し**: 「さあ、新しい一歩を踏み出そう。」<br>- **CTAボタン**: 「無料で適性診断をはじめる」 | - ファーストビューと同じデザインのCTAボタンを再度配置し、アクションを促す。 |
| **フッター** | - 運営会社情報<br>- プライバシーポリシー<br>- 利用規約<br>- **苦情処理窓口**<br>- **届け出番号** | - 小さな文字でシンプルにまとめる。<br>- 法的要件である「苦情処理窓口」と「届け出番号」を明記する。 |

## 6. その他

- **レスポンシブ対応**: スマートフォンでの閲覧がメインとなるため、モバイルファーストでデザインする。
- **表示速度**: 画像の最適化などを行い、ページの読み込み速度を高速に保つ。
